# 🍽️ Meal Finder

A simple and responsive Meal Finder web application built using **HTML, CSS, and JavaScript**. It allows users to search for meals, browse meal categories, and view detailed recipes using data from TheMealDB API.

## 🚀 Features

- 🔍 Search meals by name
- 🍲 Browse meals by category
- 📖 View complete meal details
- 🥗 Display ingredients and cooking instructions
- 📱 Responsive design for mobile and desktop
- ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- TheMealDB API

## 🌐 API Used

This project uses the free **TheMealDB API**.

Main endpoints:

- Search Meal
```
https://www.themealdb.com/api/json/v1/1/search.php?s=
```

- Meal Categories
```
https://www.themealdb.com/api/json/v1/1/categories.php
```

- Filter by Category
```
https://www.themealdb.com/api/json/v1/1/filter.php?c=
```

- Meal Details
```
https://www.themealdb.com/api/json/v1/1/lookup.php?i=
```
